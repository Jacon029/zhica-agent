// SOLUTION

基于LLM的AI采购专家Agent

核心思路：将非结构化采购需求通过大模型解析为标准字段，结合RAG技术进行多源供应商智能匹配， 最终生成可解释的推荐结论并以标准化PDF报告形式交付，全流程自动化闭环。

01

需求结构化解析

Procurement Demand Structuring

支持微信聊天、邮件、语音转写等各类非结构化输入，利用LLM精准提取品类、规格、数量、交期、预算等核心关键字段；针对信息缺失项自动发起智能追问，补全后输出标准、规范的结构化采购清单。

02

供应商智能匹配 RAG

Intelligent Supplier Screening

优先通过RAG技术检索内部250+优质供应商库；若匹配结果不足，系统自动联网抓取1688、企业官网等公开渠道数据，基于价格、交期、质量、账期建立多维度加权评分模型，输出Top3推荐结果。

03

标准化结果PDF交付

PDF Product Standardization

将筛选后的Top3供应商信息以结构化表格直观呈现，清晰罗列综合评分、单价、交期及匹配逻辑；用户确认后可一键生成合规PDF报告，支持直接归档或跨部门流转。

// SCOPE & BOUNDARY

产品定位 & 能力边界

明确定义V1.0能做什么、不能做什么，规避风险、管理预期

✅ V1.0 可实现能力

  • 非结构化需求智能识别 — 自动识别多源采购需求文本，提取关键信息并转化为标准JSON结构化数据
  • 内外部供应商智能匹配 — 优先基于RAG知识库匹配250+内部供应商，不足时自动联网补充
  • 多维评分与智能推荐 — 基于价格、交期、质量、账期等维度加权评分，自动生成Top3供应商推荐
  • 多端集成与便捷输出 — 接入飞书机器人实现IM直接调用，支持结构化表格展示与PDF一键导出
  • 输入安全与交互补全 — 智能拒识异常、涉密输入，针对缺失关键字段主动追问补全
  • 全流程可追溯管理 — 记录需求处理、供应商匹配、推荐生成的全流程日志，保障过程透明

⚠️ V1.0 不可实现边界

  • 不参与业务决策与商务承诺 — 仅提供信息服务，不可替代人工做出最终采购决策
  • 不操作核心数据与业务系统 — 不可修改、新增或删除内部供应商库数据；不直接读写ERP/OA等核心业务系统
  • 无直接外部交互与完全风险兜底 — 不直接向供应商发送询价邮件；无法保证100%无AI幻觉
  • 不直接承诺供应商价格、账期等商务条款，避免越权风险
  • 仅通过接口获取脱敏后的只读数据
  • 所有输出结果均标注"AI生成，仅供参考"，需人工复核
